Lab Exercise 1: JS Fundamentals

Goal: To practise using the fundamental elements of Javascript, including primitive and complex data types, conditionals, functions and loops.

  • Question 1: Type Coercion Analysis - Analyse 18 different JavaScript expressions to understand type conversion results
  • Question 2: String/Number Conversion Issues - Identify and fix common type conversion problems in arithmetic operations
  • Question 3: Truthiness Testing - Determine which console.log messages will print based on conditional values
  • Question 4: Ternary Operators - Rewrite if statements using conditional operators and understand += functionality
  • Question 5: Function Syntax Variations - Implement function expressions and arrow function syntax
  • Question 6: Object Methods & Context - Build Inigo Montoya object with conditional catch phrases and lastName property
  • Question 7: Method Chaining Implementation - Create basketball game with chainable methods, score tracking, and foul counting
  • Question 8: Object Property Iteration - Use for...in loops to display city information for multiple city objects
  • Question 9: Reference vs Value Assignment - Demonstrate object/array references vs primitive copying with cloning solutions
  • Question 10: Constructor Functions & Classes - Implement Person using both constructor functions and ES6 classes with canDrive method

Lab Exercise 3: JS Advanced

Goal: To practise using some advanced aspects of Javascript, including closures, decorators, applying context, asynchronous coding with timeouts and promises, prototypes, classes and error handling.

  • Question 1: Closures & Counters - makeCounter decorator with independent state and custom parameters
  • Question 2: Async Timing - delayMsg with setTimeout, execution order analysis, and clearTimeout
  • Question 3: Debouncing - Function debouncing implementation with configurable delay and arguments
  • Question 4: Fibonacci Sequences - Generate sequences using setInterval and nested setTimeout with limits
  • Question 5: Context Binding - Fix 'this' context issues in setTimeout calls with bind and cloning
  • Question 6: Function Prototypes - Add delay method to Function prototype using apply for multiple parameters
  • Question 7: Class Inheritance - PrecisionClock and AlarmClock extending DigitalClock with custom features
  • Question 8: Function Decorators - Argument validation decorators with rest operators and try-catch
  • Question 9: Promise Implementation - randomDelay promise with resolve/reject based on even/odd logic
  • Question 10: Async/Await - Convert fetch functions to async/await syntax with Promise.all for multiple URLs
